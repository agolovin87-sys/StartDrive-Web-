StartDrive

Добавьте веб-приложение на главный экран — откроется с иконкой и без адресной строки.

Открыть StartDrive

Android (Chrome)

  1. Нажмите кнопку выше или откройте сайт в Google Chrome.
  2. Всплывающее окно: нажмите Установить. Или меню ⋮ → Установить приложение / Добавить на экран «Домой».
  3. На рабочем столе появится ярлык с иконкой StartDrive.

iPhone / iPad (Safari)

  1. Откройте эту страницу или StartDrive в браузере Safari (не встроенный браузер мессенджера).
  2. Нажмите кнопку Поделиться □↑ внизу экрана.
  3. Прокрутите список и выберите На экран «Домой» → Добавить.

После установки запускайте приложение с рабочего стола — данные подгружаются с сервера, как на сайте.
