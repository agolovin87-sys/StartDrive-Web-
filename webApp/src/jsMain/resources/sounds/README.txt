Звук «Инструктор добавил свободное окно» (instruktor_dobavil_okno.mp3).

При сборке веб-приложения задача copyCadetSound копирует сюда файл:
  1) По умолчанию: из папки «Фото для приложения/звуки/instruktor _dobavil_okno.mp3»
     (путь относительно корня репозитория: ../../Фото для приложения/звуки/).
  2) Или задайте свой путь: gradlew ... -PcadetSoundSource="C:\путь\к\instruktor _dobavil_okno.mp3"
  3) Или из Android: app/src/main/res/raw/instruktor_dobavil_okno.*

Итоговое имя в этой папке: instruktor_dobavil_okno.mp3
