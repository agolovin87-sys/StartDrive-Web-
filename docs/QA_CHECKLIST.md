# Ручная проверка веб-клиента StartDrive

Краткий чеклист перед релизом. Выполнять в браузере (Chrome/Edge) после `./gradlew :webApp:jsBrowserDevelopmentRun` или деплоя.

## Авторизация и роли

- [ ] Вход email/пароль, выход.
- [ ] Админ: главная, баланс, чат, история (без ошибок в консоли).
- [ ] Инструктор: главная, запись, чат, ПДД, история.
- [ ] Курсант: главная, запись, чат, ПДД, история.

## Уведомления (админ)

- [ ] Новое событие в `admin_events` (или Cloud Function) — появляется тост и запись в списке «Уведомления».
- [ ] Повторное открытие колокольчика не дублирует одно и то же событие по id документа.

## Модальные окна

- [ ] Открыть любое модальное окно (например подтверждение выхода) — **Esc** закрывает или срабатывает «Отмена/Нет».

## ПДД

- [ ] Открыть билет, ответить на вопрос — навигация по номерам вопросов подсвечивает верно/неверно.

## Курсант

- [ ] Запись инструктора на вождение — уведомление в списке «Уведомления».
- [ ] Звук уведомления (если включён) — без сбоев.

## Мобильная вёрстка

- [ ] Узкий экран (≤400px): вкладки и шапка читаемы, без горизонтального скролла контента (кроме табов).

## Сборка

- [ ] `./gradlew :webApp:compileKotlinJs` завершается без ошибок.
